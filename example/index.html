anttech-expo-webview-bridge — web side demo

Copy this page's script (or src/web/bridgeClient.ts from the package) into your own site as the entry point for talking to the native app.

Checking bridge…

PING

Round-trips a payload to native and back. Good smoke test for the connection.

SCAN_QR

Asks native to open its QR scanner. No payload needed.

CLIPBOARD

Asks native to copy text to the device clipboard.

CALL_NUMBER

Asks native to open the phone dialer. Rate-limited on the native side.

GET_NATIVE_STORAGE

Asks native for stored data. The example native handler also *pushes* the same data unsolicited via postMessageToWeb — watch the log for both.
